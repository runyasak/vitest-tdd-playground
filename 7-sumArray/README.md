# Sum Array

สร้าง function ชื่อ `sumArray` ที่รับตัวแปร `x` ที่เป็น `number[]`
โดยนำค่าตัวเลขที่อยู่ใน array มารวมกัน และ return มาเป็น `number`

หาก array ไม่มีสมาชิกอยู่ ต้อง return `0`

## Examples

```js
  sumArray([1, 2]) -->  3
```

```js
  sumArray([5, 10, 3]) -->  18
```

```js
  sumArray([]) -->  0
```
