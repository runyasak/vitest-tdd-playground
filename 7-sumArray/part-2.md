# Sum Array (Part 2)

`sumArray` จะนำสมาชิกตัวสุดท้ายมาหารผลลัพธ์ทั้งหมด

หากตัวหารเป็น `0` ให้ return คำว่า `"Infinity"`

## Examples

```js
  sumArray([1, 2]) -->  1.5
```

```js
  sumArray([5, 10, 3]) -->  6
```

```js
  sumArray([10, 20, 10000]) -->  0.003
```

```js
  sumArray([]) -->  "Infinity"
```
